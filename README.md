# Kanban-board
Overview
This project is a Trello-style Kanban Board built using HTML, CSS, and JavaScript. It helps users organize and manage tasks by moving them between different stages of work.
Features
Create new task cards
Edit existing tasks
Delete tasks
Drag and drop cards between columns
Task priority (High, Medium, Low)
Automatic data saving using Local Storage
Responsive and user-friendly interface
Technologies Used
HTML5
CSS3
JavaScript (ES6)
Local Storage API
How It Works
Enter a task and select its priority.
Click Add Task to create a new card.
Drag and drop cards between To Do, In Progress, and Done columns.
Edit or delete tasks as needed.
All tasks are automatically saved in the browser using Local Storage and remain available after refreshing the page.
